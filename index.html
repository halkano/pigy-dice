Help

  1. Press the button with the value run to roll the dice.
  2. If the dice returns a value that is 1; player 2 rolls the dice. Else keep tossing.
  3. Press hold to double your scores and pass the dice to the next player.
  4. Whoever reaches 50 scores first wins!
